# Week 6 - Assignment 1: Basic React Setup

This directory contains the first React assignment for Week 6, focusing on getting comfortable with the React ecosystem, components, and fundamental state management.

## Overview
- **React Components:** Breaking down UI into modular, reusable functional components.
- **State Management:** Introduction to using the `useState` hook to manage local component state, enabling dynamic and interactive user interfaces.
- **JSX Syntax:** Writing UI structures using JavaScript XML (JSX) and rendering data within components.
- **Event Handling:** Capturing user inputs and interactions (like form submissions or button clicks) to update state.

## Learning Objectives
1. Understand how to scaffold a new React project using modern build tools (Vite).
2. Learn the differences between standard HTML/JS and a component-driven architecture.
3. Master the basics of the `useState` hook for data tracking.
4. Practice writing JSX, managing inline variables, and applying CSS styles in a React context




# Week 6 - Assignment 2: Advanced User Management

This directory contains the second React assignment for Week 6. It builds upon foundational React concepts by introducing more complex component structures, prop passing, and derived state.

## Overview
- **Component Composition:** Structuring an application using multiple interconnected components, such as `Users` and `Usercount`.
- **Prop Drilling:** Passing data and functions from parent components down to child components to share state across the UI tree.
- **Derived State:** Calculating values based on existing state, such as dynamically updating a user count based on an array of user objects.
- **List Rendering:** Efficiently rendering lists of data using the `.map()` method and applying unique `key` props.

## Learning Objectives
1. Understand how data flows unidirectionally (top-down) in React applications.
2. Learn how to break down complex UI into distinct, manageable components.
3. Practice communicating between parent and child components using props.
4. Enhance skills in managing arrays within state and updating them immutably.
